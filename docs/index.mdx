<img src="/assets/axol.png" width="400" alt="Axol dual-arm robot" />

Axol is a Python CLI and SDK for the Almond Axol dual-arm robot. The CLI is invoked as `axol <command> [flags]`; the SDK is importable directly from `almond_axol`.

<Note>
  **New here?** Start with the [Teleoperation quickstart](/quickstart/teleop) to go from a fresh install to a live VR teleoperation session.
</Note>
