---
title: "diag.gripper"
description: "Open then close the gripper on each arm."
---

Cycles each arm's gripper through a full open → close sweep to verify the gripper motor, calibration, and POSITION_FORCE control path. Runs the left then the right gripper and disables the motors when done.

```bash
axol diag.gripper
axol diag.gripper --no-right   # left gripper only
axol diag.gripper --no-left    # right gripper only
```

Also launchable from the [diagnostics dashboard](https://docs.almond.bot/guides/diagnostics-dashboard).
